Project Description

This project presents an interactive data visualization dashboard developed using Tableau. The objective of the project is to analyze Airbnb listings data to uncover insights related to pricing patterns, availability, demand indicators, and neighborhood-level variations. The dashboard enables users to explore different market dynamics through interactive filters and visual elements.

Dataset Description

The dataset used in this project was obtained from the Inside Airbnb open data platform. It consists of two primary CSV files: listings data and reviews data. The listings dataset includes information such as price, room type, availability, host details, and geographic coordinates, while the reviews dataset provides time-based information used to analyze demand trends and seasonality.
