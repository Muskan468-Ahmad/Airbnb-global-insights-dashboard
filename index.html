Project Description

This project presents an interactive data visualization dashboard developed using Tableau. The objective is to analyze Airbnb listings data to uncover insights related to pricing patterns, availability, demand indicators, and neighborhood-level variations. The dashboard enables users to explore market dynamics through interactive filters and visual elements.

Dataset Description

The dataset was obtained from the Inside Airbnb open data platform and consists of two primary CSV files: listings data and reviews data. The listings dataset includes pricing, room type, availability, host details, and geographic coordinates, while the reviews dataset provides time-based information used for demand and seasonality analysis.

Tools and Technologies Used

  • Tableau Public
  • GitHub Pages
  • CSV Data (Inside Airbnb)
